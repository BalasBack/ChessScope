ChessScope

Local-first tournament prep for serious players

Import your Chess.com and Lichess games, analyze with Stockfish, scout opponents, and train on your own blunders — all on your machine.

Download for Windows View on GitHub

Requires Windows 10+. First release? Build locally with npm run tauri build — see README.

Features

  • Game import from Chess.com & Lichess
  • Stockfish analysis & blunder training
  • USCF / FIDE / online opponent scouting
  • Opening stats & dashboard
  • AI coach via local Ollama (optional)

Quick start (developers)

  1. Clone the repo and run npm install
  2. Download Stockfish: scripts/download-stockfish.ps1
  3. Run npm run tauri dev
